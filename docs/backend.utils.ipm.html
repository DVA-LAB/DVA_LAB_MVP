backend.utils.ipm module

class backend.utils.ipm.IPM(camera_info, ipm_info)[소스]

기반 클래스: object

원본 프레임을 BirdEyeView (BEV)로 변환합니다. (Inverse Perspective Mapping)

BEV 변환을 위해 Pin-hole 카메라를 가정합니다.

_c: 카메라 좌표 _w: 월드 좌표 uv: 입력 이미지에 대해 원근변환 된 uv 2D 좌표

Inverse perspective mapping to a bird-eye view. Assume pin-hole camera model. There are detailed explanation of every step in the comments, and variable names in the code follow these conventions: _c for camera coordinates _w for world coordinates uv for perspective transformed uv 2d coordinates (the input image)
